# 베이 재질 이미지 2단계 편집 프롬프트

이 문서는 모든 세대의 베이 이미지에 같은 선화·조명·재질 표현 방식을 적용하기 위한 재사용 템플릿이다. 각 이미지의 형상과 배치는 원본을 따르고, 베이마다 다른 부품 구성과 색상은 `베이별 특화 규칙`으로 지정한다.

프롬프트만으로 생성형 편집 결과의 픽셀 일치를 보장할 수는 없다. 따라서 아래의 고정 프롬프트, 부품표, 베이별 규칙, 단계별 검증을 항상 함께 사용한다.

## 사용 순서

1. `작업별 입력 블록`을 작성한다. 빈 항목도 삭제하지 말고 `없음`이라고 적는다.
2. 원본과 입력 블록을 사용해 1단계만 실행한다.
3. 1단계 검증을 통과한 결과만 승인한다.
4. 승인된 1단계 결과와 고정 스타일 참고 이미지를 사용해 2단계를 실행한다.
5. 공통 검증과 베이별 실패 조건을 모두 확인한다.
6. 실패하면 전체 프롬프트를 바꾸지 말고 `단일 항목 수정 프롬프트`로 한 항목만 다시 편집한다.

1단계와 2단계를 한 요청으로 합치지 않는다.

## 작업별 입력 블록

아래 블록을 베이마다 작성해 각 단계의 `{{WORK_CONFIG}}` 위치에 그대로 넣는다.

```text
[기본 정보]
베이 이름: {{BEY_NAME}}

[공통 재질 부품표]
금속 부품: {{METAL_PARTS}}
클리어 부품과 기본색:
- {{CLEAR_PART_1}} = {{CLEAR_HEX_1}}
- {{CLEAR_PART_2}} = {{CLEAR_HEX_2}}
불투명 플라스틱: {{OPAQUE_PARTS}}
고무·특수 재질: {{SPECIAL_PARTS}}
스티커·도색·인쇄: {{PRINTED_PARTS}}
실제 구멍·빈 공간: {{HOLES}}

[베이별 특화 규칙]
부품 범위 확장: {{PART_EXTENSIONS}}
공통 재질 분류 예외: {{MATERIAL_EXCEPTIONS}}
1단계 전용 제거 대상: {{STAGE1_SPECIAL_REMOVALS}}
2단계 전용 재질·조명 규칙: {{STAGE2_SPECIAL_RENDERING}}
반드시 원본 그대로 보존할 요소: {{SPECIAL_PRESERVATION}}
베이별 실패 조건: {{BEY_SPECIFIC_FAILURES}}
```

클리어 부품이 여러 색이면 부품별로 항목을 추가한다. 클리어 부품이 없으면 `클리어 부품과 기본색: 없음`으로 적고 관련 규칙을 적용하지 않는다. 목록에 없는 부품의 재질을 임의로 추정하거나 변경하지 않는다.

## 규칙 적용 우선순위

모든 단계에서 다음 순서를 적용한다.

1. 원본의 크기·형상·선화·배치·알파 보존
2. 베이별 특화 규칙
3. 입력된 부품별 재질표와 HEX 색상
4. 공통 재질·조명 규칙
5. 스타일 참고 이미지

조건이 충돌하면 높은 순위의 조건을 따른다. 스타일 참고 이미지 때문에 원본 형상이나 베이별 색상·재질 분류를 변경하지 않는다.

## 1단계: 기존 조명 제거

```text
Use case: precise-object-edit
Asset type: 베이스타디움 도감용 베이 정면 제품 이미지

[입력 이미지]
이미지 1은 실제 편집 대상인 원본 베이 이미지다. 이미지 1은 형태, 선화, 부품 배치, 스티커, 구도, 캔버스와 알파 채널의 절대 기준이다.

[작업별 설정]
{{WORK_CONFIG}}

[작업 목표]
이미지 1의 기존 조명·반사·그림자만 제거해 조명 없는 플랫 재질 기준본을 만든다. 새 이미지를 만들거나 베이를 다시 그리지 않는다.

[규칙 적용 우선순위]
1. 원본의 크기·형상·선화·배치·알파 보존
2. 베이별 특화 규칙
3. 입력된 부품별 재질표와 HEX 색상
4. 아래 공통 처리 규칙

[절대 고정 조건]
- 원본과 동일한 캔버스 크기, 피사체 크기, 위치, 중심점, 회전, 여백과 투명 배경을 유지한다.
- 형태, 실루엣, 비율, 두께, 홈, 구멍, 톱니, 부품 경계, 윤곽선과 내부 선화를 변경하지 않는다.
- 스티커, 로고, 인쇄 문양, 글자와 색 배치를 원본 그대로 유지한다.
- 검은색 또는 짙은 색 구조선은 그림자가 아니므로 삭제하거나 흐리게 하지 않는다.
- 투명 픽셀과 반투명 가장자리의 알파를 변경하지 않는다.
- 부품을 재해석·단순화·보정하거나 원본에 없는 면과 디테일을 만들지 않는다.

[제거 대상]
- 기존 하이라이트, 반사광, 광택과 흰색 번쩍임
- 밝고 어두운 조명 그라데이션과 방향성 그림자
- 어두운 반사면, 색 번짐, 후광과 국소적인 빛 얼룩
- 베이별 특화 규칙에 지정된 추가 오염

[공통 재질 처리]
- 금속 부품: 기존 조명과 금속 반사를 제거하고 원본 중간톤에서 추출한 균일한 기본 금속색으로 정리한다.
- 클리어 부품: 부품표에 입력된 HEX를 조명 없는 기본 중간톤으로 사용한다. 서로 다른 클리어 부품의 HEX를 섞지 않는다.
- 클리어 부품에 존재하는 흰색, 은색, 밝은 회색과 중립 회색 조명은 모두 제거 대상이다.
- 아래 금속이 클리어 부품을 통해 비치는 부분은 중립 회색 그대로 남기지 말고 해당 클리어 HEX로 착색된 균일한 투과색으로 정리한다.
- 불투명 플라스틱과 고무: 원본의 대표 중간색을 유지한 균일한 무광 기본색으로 정리한다.
- 스티커·도색·인쇄: 원본 픽셀을 유지하고 색상, 밝기, 글자와 경계를 변경하지 않는다.
- 실제 구멍과 빈 공간은 채우지 않는다.

[평탄화 방식]
- 각 부품과 각 면의 기존 경계 안에서만 평탄화한다.
- 서로 다른 면이나 부품을 하나의 색 영역으로 합치지 않는다.
- 블러로 선, 작은 홈, 톱니와 안티앨리어싱을 뭉개지 않는다.

[금지]
새 하이라이트, 새 그림자, 새 그라데이션, 재질 강화, 발광, 후광, 입자, 노이즈, 얼룩, 브러시 질감, 형상 보정, 디테일 추가, 새로운 부품 생성을 금지한다.

[출력]
원본과 정확히 같은 크기, 피사체 배치와 알파 채널을 가진 PNG. 변경 범위는 원본의 불투명·반투명 피사체 내부에 한정한다.
```

## 2단계: 공통 스타일의 조명·재질 적용

```text
Use case: precise-object-edit
Asset type: 베이스타디움 도감용 베이 정면 제품 이미지

[입력 이미지]
- 이미지 1: 승인된 1단계 결과. 형태, 선화, 부품 배치, 색상, 스티커, 캔버스와 알파의 절대 기준이 되는 실제 편집 대상이다.
- 이미지 2: 모든 베이에 공통으로 사용하는 재질 스타일 참고 이미지다. 조명 강도, 선화 중심 표현과 재질 대비만 참고한다.

이미지 2의 베이 형상, 부품 구성, 비율, 색상, 스티커, 글자, 구도와 배경은 복사하지 않는다.

[작업별 설정]
{{WORK_CONFIG}}

[작업 목표]
이미지 1의 구조와 부품별 기본색을 유지하면서 모든 베이에 공통된 애니메이션 60%·정돈된 3D 제품 렌더 40% 스타일의 정면 재질 조명을 적용한다.

[규칙 적용 우선순위]
1. 이미지 1의 크기·형상·선화·배치·알파 보존
2. 베이별 특화 규칙
3. 입력된 부품별 재질표와 HEX 색상
4. 아래 공통 재질·조명 규칙
5. 이미지 2의 스타일

[절대 고정 조건]
- 이미지 1과 동일한 캔버스 크기, 피사체 크기, 위치, 중심점, 회전, 여백, 실루엣과 알파 채널을 유지한다.
- 형태, 비율, 두께, 홈, 구멍, 톱니, 부품 경계, 윤곽선과 내부 선화를 변경하지 않는다.
- 스티커, 로고, 인쇄 문양과 글자를 원본 그대로 유지한다.
- 기존 구조에서 확인되지 않는 면, 홈, 장식과 부품을 새로 만들지 않는다.
- 투명 배경에 그림자, 반사, 색, 흰색 배경과 체크무늬를 추가하지 않는다.

[공통 스타일]
- 원본의 또렷한 애니메이션 선화를 재질 효과보다 우선해 유지한다.
- 재질과 입체감은 깨끗하고 절제된 3D 제품 일러스트 수준으로 표현한다.
- 사진처럼 거친 표면 흠집이나 실사형 PBR 질감을 사용하지 않는다.
- 표면은 깨끗하며 입자, 노이즈, 얼룩, 브러시 흔적과 흐린 구름무늬가 없어야 한다.

[공통 조명]
- 카메라와 같은 위치의 넓고 부드러운 정면 중앙 광원만 사용한다.
- 조명장은 이미지 중심을 기준으로 좌우·상하 대칭이어야 한다.
- 한쪽 방향의 하이라이트, 비대칭 그림자와 국소적인 빛 얼룩을 만들지 않는다.
- 원본 굴곡과 부품 경계 안에서만 하이라이트·중간톤·어두운 면을 형성한다.
- 반사 대비는 금속 > 클리어 > 불투명 플라스틱 > 스티커 순서로 표현한다.

[금속 부품]
- 원본이 무도색 금속이면 밝고 깨끗한 은회색 아연합금으로 표현한다. 도색 금속이면 원본 색조를 유지한다.
- 넓고 매끄러운 밝은 반사면과 절제된 어두운 반사면을 함께 적용해 금속성이 명확하게 보여야 한다.
- 실제 모서리와 베벨에만 얇고 선명한 금속 하이라이트를 적용한다.
- 금속 반사 대비는 클리어와 불투명 플라스틱보다 확실히 강해야 한다.
- 거친 입자, 점상 노이즈, 방향성 브러시, 거울 같은 크롬 반사와 과도한 순백색 번쩍임을 금지한다.

[클리어 부품]
- 부품표에 입력된 HEX를 넓은 면의 기본 중간톤으로 유지한다.
- 밝은 면, 어두운 면, 가장자리 집중색과 투과색도 해당 HEX에서 파생된 같은 색 계열만 사용한다.
- 흰색, 은색, 밝은 회색과 중립 회색의 하이라이트·반사·번짐을 사용하지 않는다.
- 반사 강도와 반사 면적은 금속보다 확실히 낮게 한다.
- 외곽과 실제 돌출부에만 얇고 절제된 유색 반사를 적용한다.
- 두꺼운 가장자리 안쪽에는 같은 색 계열의 좁고 진한 색 집중대를 적용한다.
- 아래 금속이 비치는 곳은 해당 클리어 HEX로 착색된 금속색과 반사만 균일하게 보이게 한다.
- 불투명 단색, 형광색 테두리, 넓은 후광, 검은 반사띠, 흰색 발광선과 유리 가루 같은 노이즈를 금지한다.

[기타 부품]
- 불투명 플라스틱과 고무에는 원래 색상을 유지한 부드러운 셀 명암만 적용하고 금속 반사를 넣지 않는다.
- 스티커·도색·인쇄는 원본 색상과 글자를 유지한 무광 평면으로 표현한다.
- 실제 구멍과 빈 공간은 채우지 않는다.

[베이별 특화 규칙 적용]
- 특화 규칙에 포함된 부품 범위와 재질 예외를 공통 규칙보다 우선한다.
- 특화 규칙의 보존 요소는 재질 효과 아래에서 원본과 동일하게 유지한다.
- 특화 규칙의 실패 조건에 해당하는 결과를 만들지 않는다.

[출력]
이미지 1과 정확히 같은 크기, 피사체 배치와 알파 채널을 가진 PNG.
```

## 단일 항목 수정 프롬프트

검증에서 한 항목이 실패했을 때 다음 프롬프트로 해당 부분만 다시 편집한다.

```text
Use case: precise-object-edit

현재 이미지는 승인 후보이며 아래 실패 항목 하나만 수정한다.

실패 항목: {{FAILED_RULE}}
수정할 부품과 위치: {{TARGET_PART_AND_LOCATION}}
정확한 수정 내용: {{TARGETED_CORRECTION}}

이 수정 외에는 어떤 픽셀도 의도적으로 변경하지 않는다. 현재 이미지의 캔버스, 피사체 크기, 위치, 실루엣, 알파, 선화, 홈, 구멍, 스티커, 글자, 다른 부품의 색상과 재질 효과를 그대로 유지한다. 부품을 다시 그리거나 형상을 재해석하지 않는다.
```

## 리브라 작업별 입력 예시

```text
[기본 정보]
베이 이름: 리브라

[공통 재질 부품표]
금속 부품: 외곽의 네 방향 메탈휠, 클리어휠 아래로 보이는 회색 구조 프레임, 중앙 스티커에 인접한 회색 직사각형 연결부
클리어 부품과 기본색:
```
